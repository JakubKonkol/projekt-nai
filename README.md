# Projekt na przedmiot narzędzia sztucznej inteligencji

## Prerequisites 📋
- Python 3.9
- Node.js
## Building and running the project 🛠️
1. `git clone https://github.com/JakubKonkol/projekt-nai.git`
2. `cd projekt-nai`
3. `pip install -r requirements.txt`
4. `python -m flask run`
5. `cd frontend` 
6. `npm install` 
7. `ng serve --open`
